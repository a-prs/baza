# Установка Claude Code

Claude Code — самый мощный ИИ-инструмент для вайб-кодинга. В этой главе установим его, настроим и подготовим к работе.

## Что нужно перед установкой

- ✅ VS Code установлен
- ✅ Node.js установлен (версия 18+)
- ✅ Подписка Claude Pro ($20/мес) или Max ($100/мес)

::: info Про подписку
Claude Code требует платную подписку Anthropic. Если ты хочешь попробовать вайб-кодинг бесплатно — используй [Qwen Code](/practice/00-quickstart) (1000 запросов/день бесплатно). Claude Code — это следующий уровень, когда ты уже попробовал и хочешь больше мощности.
:::

## Шаг 1: Оформи подписку

1. Перейди на **[claude.ai](https://claude.ai/)**
2. Зарегистрируйся или войди
3. Нажми **Upgrade to Pro** (или Max)
4. Оплати подписку

**Pro ($20/мес)** — хватает для большинства задач. **Max ($100/мес)** — больше запросов и приоритет, для тех, кто работает с Claude Code интенсивно каждый день.

## Шаг 2: Установи Claude Code CLI

Открой терминал в VS Code и введи:

```bash
npm install -g @anthropic-ai/claude-code
```

Дождись окончания установки. Проверь:

```bash
claude --version
```

Если видишь номер версии — CLI установлен.

## Шаг 3: Авторизуйся

```bash
claude
```

При первом запуске Claude Code откроет браузер для авторизации. Войди в свой аккаунт Anthropic. После успешного входа терминал покажет, что ты авторизован.

## Шаг 4: Установи расширение для VS Code

Работать с Claude Code можно двумя способами:
- **Через терминал** — пишешь `claude` и общаешься
- **Через расширение VS Code** — боковая панель с чатом (удобнее)

Устанавливаем расширение:

1. В VS Code нажми `Ctrl+Shift+X`
2. В поиске напиши **Claude Code**
3. Установи официальное расширение от **Anthropic**
4. Слева появится иконка Claude — это твоя панель для общения с ИИ

Расширение использует тот же CLI под капотом, просто обёрнуто в красивый интерфейс.

## Шаг 5: Настрой глобальный CLAUDE.md

Это файл с правилами, которые Claude Code будет соблюдать **во всех проектах**.

Открой терминал и создай файл:

**Windows:**
```bash
mkdir %USERPROFILE%\.claude 2>nul
notepad %USERPROFILE%\.claude\CLAUDE.md
```

**Mac / Linux:**
```bash
mkdir -p ~/.claude
nano ~/.claude/CLAUDE.md
```

Вставь туда базовые правила:

```markdown
# Глобальные правила

## Язык
- Общайся со мной на русском
- Код и комментарии в коде — на английском

## Безопасность
- Не удаляй файлы без моего разрешения
- Перед значительными изменениями — спроси подтверждение
- Не перезаписывай данные без спроса
- Не коммить в git без моей команды

## Стиль работы
- Не добавляй фичи, о которых я не просил
- Не рефактори код без просьбы
- Если видишь ошибку — скажи, но не чини без спроса
- Пиши простой, понятный код без переусложнений
```

Сохрани файл. Теперь Claude Code будет следовать этим правилам в каждом проекте.

## Шаг 6: Создай первый проект с CLAUDE.md

Создадим правильную структуру проекта:

```bash
mkdir Projects/my-project
cd Projects/my-project
```

Создай проектный `CLAUDE.md`:

```bash
claude
```

И скажи ему:

```
Создай файл CLAUDE.md в корне проекта со следующим содержанием:

# Мой проект

## О проекте
Одностраничный сайт-визитка.

## Стек
- HTML, CSS, JavaScript
- Без фреймворков

## Правила
- Тёмная тема
- Адаптивность под мобильные
- Весь код в одном файле index.html
```

Claude Code создаст файл. Теперь при каждом запуске в этой папке он будет читать `CLAUDE.md` и понимать контекст проекта.

## Настройки разрешений

Claude Code спрашивает разрешение на каждое действие: создать файл, запустить команду, изменить код. Это безопасно, но иногда утомляет.

Ты можешь настроить уровень доверия:

- **Ask** (по умолчанию) — спрашивает всё
- **Auto-edit** — автоматически редактирует файлы, но спрашивает перед запуском команд
- **YOLO** — делает всё без спроса (не рекомендуется для новичков)

Для начала оставь **Ask** — так ты будешь видеть всё, что ИИ делает, и учиться.

## Полезные команды Claude Code

| Команда | Что делает |
|---------|-----------|
| `claude` | Запустить Claude Code в текущей папке |
| `claude "задача"` | Запустить с конкретной задачей |
| `/help` | Показать справку |
| `/clear` | Очистить контекст (начать «с чистого листа») |
| `/cost` | Показать, сколько потрачено токенов |
| `Ctrl+C` | Прервать выполнение |
| `Esc` | Выйти из Claude Code |
