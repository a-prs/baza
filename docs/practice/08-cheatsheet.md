# Шпаргалка

## Глоссарий

| Термин | Что это |
|--------|---------|
| **API** | Способ, которым программы общаются друг с другом |
| **CLI** | Программа, которая работает через терминал (текстовые команды) |
| **CLAUDE.md** | Файл с правилами и контекстом для ИИ |
| **CODE_MAP.md** | Карта проекта — какой файл за что отвечает |
| **Деплой** | Выложить проект на сервер, чтобы он был доступен в интернете |
| **DNS** | Система, которая переводит домен (имя) в IP-адрес (числа) |
| **Домен** | Красивый адрес сайта (например, mysite.com) |
| **Git** | Система контроля версий — «машина времени» для файлов |
| **GitHub** | Онлайн-хранилище Git-проектов |
| **HTTPS/SSL** | Защищённое соединение (замочек в браузере) |
| **IDE** | Программа для написания и редактирования кода (например, VS Code) |
| **IP-адрес** | Числовой адрес сервера в интернете |
| **Low-code** | Создание проектов с минимумом кода (n8n, Retool) |
| **MCP** | Протокол подключения ИИ к внешним сервисам |
| **No-code** | Создание проектов без кода (Tilda, Notion) |
| **Node.js** | Среда для запуска JavaScript вне браузера |
| **n8n** | Платформа автоматизаций (связывает сервисы в цепочки) |
| **SSH** | Удалённое подключение к серверу через терминал |
| **Терминал** | Окно для текстовых команд компьютеру |
| **VPS** | Виртуальный сервер, который ты арендуешь |
| **VS Code** | Бесплатная IDE от Microsoft — «швейцарский нож» кодера |

## Частые ошибки новичков

### «Команда не найдена» (command not found)
**Причина:** программа не установлена или не добавлена в PATH.
**Решение:** переустанови программу, убедись что поставил галочку «Add to PATH».

### «Нет прав» (permission denied)
**Причина:** нужны права администратора.
**Решение:** на сервере — добавь `sudo` перед командой. На Windows — запусти терминал от имени администратора.

### «Порт занят» (port already in use)
**Причина:** другая программа уже использует этот порт.
**Решение:** останови другую программу или смени порт.

### ИИ делает не то, что я прошу
**Причина:** промпт слишком размытый.
**Решение:** будь конкретнее. Вместо «сделай красиво» → «сделай тёмный фон #1a1a2e, белый текст, шрифт Inter 16px, кнопки со скруглением 8px».

### ИИ сломал то, что работало
**Причина:** ИИ внёс слишком много изменений за раз.
**Решение:** используй Git (`git diff` — посмотреть что изменилось, `git checkout .` — откатить все изменения). Давай ИИ задачи маленькими порциями.
