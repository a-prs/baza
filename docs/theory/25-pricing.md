# Ценообразование AI-продуктов

Сколько брать денег — один из самых сложных вопросов. Большинство indie-разработчиков берут слишком мало. Разбираем подходы от первого платного пользователя до зрелого продукта.

## Три подхода

| Подход | Суть | Когда применять |
|--------|------|-----------------|
| Cost-plus | себестоимость × N | Никогда — не учитывает ценность |
| Competitor-based | как конкуренты | Старт (защититься от «слишком дорого») |
| Value-based | сколько стоит клиенту проблема | Зрелый продукт, B2B |

**Правило:** начинай с competitor-based → переходи на value-based как только понял что покупает клиент.

## Структура цен для AI-продуктов

### Freemium

Бесплатный лимит → платный безлимит:

```
Бесплатно: 10 запросов/день
Pro: 999₽/мес — безлимит
```

**Работает для:** инструментов где ROI не очевиден до использования (клиент должен попробовать).

**Не работает для:** B2B-решений (бесплатные пользователи создают издержки без конверсии).

Принцип: бесплатный лимит должен давать «вау-момент» но не закрывать основной use case полностью.

### Flat rate (единая цена)

Одна цена за всё:

```
499₽/мес — любые функции, любой объём
```

**Работает для:** узкие инструменты с понятным применением, где объём не важен.

**Минус:** не учитывает сегменты — стартапы платят как корпорации.

### Тиерированная подписка

```
Стартер:  499₽/мес   — 1 бот, 1 000 запросов/мес
Бизнес:  1 499₽/мес  — 5 ботов, 10 000 запросов/мес
Про:     3 999₽/мес  — безлимит, API-доступ, поддержка
```

**Правило 3 тиеров:** дешёвый (снижает барьер), средний (якорь — купит большинство), дорогой (anchor для «среднего»).

### Per-use (оплата за использование)

```
50₽ за 1 000 слов транскрипции
5₽ за 1 изображение
```

**Работает для:** нерегулярных задач, где подписка неудобна.

**Минус:** сложно предсказать доход; нужна система пополнения баланса.

## Сколько брать (ориентиры для РФ)

```
B2C инструмент / чат-бот:     299–999₽/мес
B2C нишевой (юрист, врач):    999–2 999₽/мес
B2B малый бизнес:            2 000–9 999₽/мес
B2B средний / enterprise:   10 000–50 000₽/мес
```

**Правило первой цены:** возьми цифру которая кажется «страшно дорого» → дели на 2. Это твоя стартовая цена. Потом повышай.

## Себестоимость AI (не ошибись)

Для Claude Haiku (~$0.25/1M токенов):

```
Типичный запрос пользователя:
- Входной промпт: 500 токенов
- Ответ: 300 токенов
- Итого: 800 токенов ≈ $0.0002 = 0.02₽

1000 запросов в день:
- Токены: 0.02₽ × 1000 = 20₽/день = 600₽/мес
- Сервер VPS: 1 000₽/мес
- Итого себестоимость: ~1 600₽/мес на 30 пользователей
```

При тарифе 499₽/мес × 30 пользователей = 14 970₽/мес выручка. Маржа ~89%. Типична для AI SaaS.

**Важно:** посчитай стоимость **самого дорогого сценария** (не среднего). Если кто-то загрузит 100-страничный PDF — сколько это токенов?

## Годовая vs месячная подписка

```
Месяц:  999₽/мес
Год:    7 990₽/год (скидка ~33%, = 666₽/мес)
```

**Зачем давать годовую скидку:**
- Ты: получаешь деньги сейчас (cash flow), снижаешь churn
- Клиент: платит меньше в пересчёте на месяц

Правило: скидка 20-40% на год. Меньше — не стимулирует. Больше — теряешь маржу.

## Когда повышать цены

Сигналы что пора:
1. Конверсия из пробного в платный > 8-10% — не слишком дорого
2. Churn < 5%/мес — продукт ценен
3. Пользователи сами говорят «дёшево» или рекомендуют без бонусов
4. Очередь входящих / лист ожидания

Как повышать:
- Дедлайн: «цена вырастет 1 августа»
- Дедлайн + гарантия: новая цена — только для новых, текущие локируют старую навсегда
- Тест A/B: половине показываешь +30%, смотришь на конверсию

## Триал vs Freemium vs Money-back

| Модель | Барьер входа | Конверсия | Подходит |
|--------|-------------|-----------|----------|
| Freemium | Нет | 2-5% | Массовый B2C |
| Триал 7-14 дней | Низкий | 15-25% | B2B, дорогие тарифы |
| Money-back 30 дней | Нужна карта | 20-35% | Высокая уверенность в продукте |
| Без триала | Высокий | — | Нишевые B2B с демо |

**Лучшая модель для старта:** freemium (снижает барьер) + ограниченный функционал (мотивирует апгрейд). Триал только если умеешь онбордить за 3-5 дней.

## Психология цены

```
Цены которые работают:
499 вместо 500  (psychological pricing)
990 вместо 1000
4 990 вместо 5 000

Что НЕ делать:
- Цена 1₽ в пробном → потом 999₽ (потеря доверия)
- Скрытые лимиты в описании (пользователь злится)
- Слишком много тиеров (>4 → паралич выбора)
```

## Пример расчёта для Telegram-бота

```
Продукт: AI-помощник для юристов

ЦА: адвокаты и юристы малого бизнеса
Проблема: составление документов, поиск по законодательству
Альтернатива: 1-2 часа ручной работы × 3 000₽/час = 3-6 тыс за документ

Value-based цена:
- Бот экономит 1-2 часа в день → 3 000₽/день экономии
- Брать 10% от экономии = 300₽/день = 9 000₽/мес

Реальная цена (с учётом психологии и рынка):
- Стартер: 990₽/мес  — 50 запросов
- Про:     2 490₽/мес — безлимит + приоритет
- Команда: 5 990₽/мес — 5 пользователей

Тест: 10 первых клиентов с ценой 990₽ →
если покупают без вопросов — поднять до 1 490₽
```

---

::: tip Первая продажа — вручную
Не уходи в автоматизацию пока не продал вручную 5-10 раз. Ручные продажи дают: реальную реакцию на цену, понимание возражений, историю для страницы отзывов. Только потом страница оплаты + автоматизация.
:::
