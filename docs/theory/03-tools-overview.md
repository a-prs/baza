# Обзор инструментов

Теперь, когда ты знаешь термины и понимаешь суть вайб-кодинга, давай посмотрим на конкретные инструменты. Что ставить, чем пользоваться и сколько это стоит.

## VS Code — твой рабочий стол

**Visual Studio Code** (VS Code) — бесплатная программа от Microsoft, в которой ты будешь работать большую часть времени. Это IDE (среда разработки), но для вайб-кодинга она выполняет три ключевые роли:

1. **Редактор файлов** — тут ты видишь файлы своего проекта, можешь их открывать и читать
2. **Терминал** — встроен прямо внутрь, не нужно отдельное окно
3. **Расширения** — можно установить плагины для ИИ-кодинга (Claude Code, Copilot и другие)

::: tip Проще говоря
VS Code — это швейцарский нож вайб-кодера. Один инструмент, в котором есть всё: и файлы посмотреть, и команды написать, и ИИ подключить. Не нужно прыгать между десятью окнами.
:::

## Расширения для VS Code — ИИ прямо в редакторе

VS Code умеет устанавливать **расширения** (extensions) — это как приложения для телефона, только для редактора кода. Через расширения можно подключить ИИ-помощника прямо в VS Code, без отдельных программ.

::: tip Проще говоря
VS Code из коробки — это смартфон без приложений. Расширения — это приложения, которые ты ставишь из магазина. Поставил расширение для ИИ — и у тебя в редакторе появился умный помощник, который пишет код.
:::

### Claude Code (расширение)

Официальное расширение от Anthropic. Ты открываешь панель в VS Code, общаешься с Claude текстом — он видит твой проект, редактирует файлы, запускает команды. Именно так мы работаем прямо сейчас, когда ты это читаешь — этот гайд написан через Claude Code в VS Code.

**Стоимость:** нужна подписка Claude Pro ($20/мес) или Max ($100/мес).

### GitHub Copilot (расширение)

Расширение от Microsoft/GitHub. Подсказывает код по мере набора, может генерировать функции и объяснять код. Есть бесплатный тариф с ограничениями.

### Gemini Code Assist (расширение)

Расширение от Google. Подключает ИИ Gemini к VS Code. Умеет генерировать код, объяснять, рефакторить.

### Qwen Code (расширение + CLI)

::: info Бесплатный вайб-кодинг
**[Qwen Code](https://github.com/QwenLM/qwen-code) — это сейчас самый простой способ начать вайб-кодинг бесплатно.** Это open-source аналог Claude Code от Alibaba. Работает и как CLI в терминале, и как расширение для VS Code, Zed и JetBrains. Бесплатно — **1000 запросов в день** через Qwen OAuth. Не нужны никакие подписки и кредитные карты.

Подробная инструкция — в практической главе «Самый быстрый старт».
:::

## CLI-инструменты — ИИ в терминале

CLI-инструменты — это самый мощный способ вайб-кодить. Ты запускаешь их в терминале, описываешь задачу обычным текстом, а ИИ сам создаёт файлы, запускает команды, чинит ошибки. Ниже — два главных варианта.

### Claude Code (CLI)

**Claude Code** — CLI от Anthropic (компании, которая делает ИИ Claude). Самый мощный инструмент на рынке.

**Почему Claude Code:**
- Работает с любым проектом (сайты, боты, скрипты, приложения)
- Понимает контекст — видит все файлы проекта и понимает, что к чему
- Умеет сам запускать команды, устанавливать зависимости, чинить ошибки
- Можно настроить правила поведения через файл `CLAUDE.md` (об этом — отдельная глава)

**Стоимость:** нужна подписка Claude Pro ($20/мес) или Max ($100/мес). Pro хватает для начала.

### Qwen Code (CLI)

**[Qwen Code](https://github.com/QwenLM/qwen-code)** — open-source CLI от Alibaba. По функциональности очень похож на Claude Code: живёт в терминале, видит проект, создаёт файлы, запускает команды. Оптимизирован под модель Qwen3-Coder.

**Почему Qwen Code:**
- **Бесплатно** — 1000 запросов в день через Qwen OAuth (без кредитной карты)
- Open-source — полностью открытый код
- Работает как CLI и как расширение для VS Code / JetBrains / Zed
- Умеет подключаться к другим API (OpenAI, Anthropic, Gemini)

**Стоимость:** бесплатно (1000 запросов/день). Можно подключить платные API для других моделей.

## Cursor — альтернатива «всё в одном»

**Cursor** — это, по сути, VS Code, но с уже встроенным ИИ. Не нужно ничего дополнительно устанавливать — просто открываешь Cursor и начинаешь общаться с ИИ прямо в редакторе.

**Плюсы:**
- Быстрый старт — скачал и работаешь
- Красивый интерфейс для общения с ИИ
- Видит твой код и может его редактировать

**Минусы:**
- Платная подписка ($20/мес)
- Менее гибкий, чем Claude Code, для сложных задач

::: tip Проще говоря
Claude Code — это нанять подрядчика, которому ты даёшь задания по телефону (через терминал). Cursor — это подрядчик, который сидит рядом с тобой за столом и видит, на что ты показываешь пальцем. Оба работают хорошо, но по-разному.
:::

## Другие варианты

### Windsurf (бывший Codeium)
Ещё одна IDE со встроенным ИИ, похожая на Cursor. Есть бесплатный тариф с ограничениями. Хороший вариант попробовать вайб-кодинг без оплаты.

### GitHub Copilot
ИИ-плагин от Microsoft/GitHub. Встраивается в VS Code. Хорошо дописывает код, но менее самостоятельный, чем Claude Code — больше помогает, чем делает за тебя.

### Bolt и Lovable
Работают прямо в браузере. Не нужно ничего устанавливать. Ты описываешь задачу — получаешь готовый сайт или приложение. Идеальны для быстрого прототипа, но ограничены в возможностях.

### Replit Agent
Онлайн-платформа для кодинга с ИИ-агентом. Всё в браузере — написал, запустил, задеплоил. Удобно для новичков, но платно для серьёзного использования.

## Termius — для работы с серверами

**Termius** — это красивая и удобная программа для подключения к серверам по SSH. Работает на Windows, Mac, Linux и даже на телефоне.

**Зачем нужна:** когда у тебя появится свой сервер (VPS), тебе нужно будет к нему подключаться и вводить команды. Termius делает это удобным:

- Сохраняет все твои серверы в списке — не нужно каждый раз вводить адрес и пароль
- Красивый терминал с нормальными шрифтами
- Работает на телефоне — можешь подключиться к серверу даже с дивана
- Есть SFTP — передача файлов между твоим компьютером и сервером

::: tip Проще говоря
Termius — это записная книжка с адресами серверов и пульт управления ими. Вместо того чтобы каждый раз вспоминать «а какой там IP у моего сервера и какой пароль», ты один раз сохранил — и потом подключаешься одним кликом.
:::

## n8n — автоматизации без кода

**n8n** — это платформа для создания автоматизаций. Ты соединяешь блоки (ноды) в цепочку, и n8n выполняет их один за другим.

**Примеры:**
- «Каждый день в 9 утра собери новости из RSS и пришли мне в Telegram»
- «Когда приходит письмо на почту — сохрани вложение в Google Drive»
- «Когда кто-то заполняет форму — создай задачу в Notion»

**Почему n8n, а не Zapier или Make:**
- Можно поставить на свой сервер (бесплатно, без лимитов)
- Открытый код — ничего не прячут
- 800+ готовых интеграций
- Есть ИИ-ноды для работы с нейросетями

::: tip Проще говоря
n8n — это как конвейер на заводе. Ты один раз настраиваешь: «сначала деталь сюда, потом туда, потом покрасить, потом упаковать». И дальше конвейер работает сам, 24/7, без твоего участия.
:::

## Low-code платформы для деплоя

Если ты не хочешь (или пока не готов) возиться с серверами, есть платформы, которые берут всё на себя:

### Vercel
Загружаешь проект → он автоматически появляется в интернете. Идеально для сайтов и веб-приложений. Бесплатный тариф — хватает для личных проектов.

### Netlify
Почти то же самое, что Vercel. Загружаешь файлы → получаешь ссылку. Бесплатно для базового использования.

### Supabase
База данных + авторизация + хранилище файлов. Если твоему проекту нужно хранить данные (пользователей, заказы, записи) — Supabase решает это без настройки сервера.

::: tip Проще говоря
Свой сервер (VPS) — это арендовать пустой офис и обставлять его самому. Vercel/Netlify — это коворкинг: пришёл, сел, работаешь. Быстро и просто, но меньше контроля. Supabase — это готовый складской комплекс: просто привози и забирай коробки (данные), об остальном позаботятся.
:::
