# Бизнес-модели: как зарабатывать на продуктах

Написать код — полдела. Вопрос «как на этом зарабатывать» часто важнее технической реализации. Разберём модели которые работают для вайб-кодеров.

## Модель 1: Консалтинг и заказная разработка

Самый быстрый путь к деньгам. Клиент платит за твоё время и экспертизу.

**Что продаёшь:** внедрение AI-инструментов, автоматизация процессов, разработка ботов и инструментов под задачи клиента.

**Ценообразование:**
- Почасовое: $50–150/ч (РФ: 3000–8000 руб/ч)
- За проект: фиксированная сумма после оценки
- Ретейнер: ежемесячная плата за поддержку и развитие

**Плюсы:** деньги сразу, не нужен трафик, высокий чек.  
**Минусы:** время ≠ масштаб, зависимость от клиентов, «день не работаешь — день не ешь».

**Как начать:** сделай одно реальное кейс-исследование (пусть даже для себя или «за портфолио»), опиши проблему → решение → результат → прайс. Публикуй, показывай людям в нише.

## Модель 2: SaaS (подписка)

Делаешь продукт один раз — получаешь деньги каждый месяц.

**Что продаёшь:** доступ к инструменту: бот, веб-приложение, API.

**Unit economics:**
- MRR (Monthly Recurring Revenue) = кол-во подписчиков × цена
- LTV (Lifetime Value) = MRR / churn (% отписок в месяц)
- CAC (Customer Acquisition Cost) = траты на маркетинг / новые клиенты
- Бизнес работает когда LTV > 3×CAC

**Пример:** 200 подписчиков × 990 руб/мес = 198 000 руб/мес. Churn 5% → 1 месяц в среднем у каждого. LTV = 990/0.05 = 19 800 руб.

**Плюсы:** предсказуемый доход, масштабируется.  
**Минусы:** долго строить базу, нужна поддержка, churn убивает бизнес.

**Когда считать:** 100 платных подписчиков = момент когда стоит вкладываться серьёзнее.

## Модель 3: Контент + партнёрки

Создаёшь аудиторию через контент, монетизируешь рекламой и реферальными программами.

**Что продаёшь:** внимание аудитории (реклама) или рекомендуешь инструменты (партнёрки).

**Партнёрки AI-инструментов:**
- Anthropic: нет публичной программы
- OpenAI: нет
- Cursor: реферальная программа
- DigitalOcean: $25 за каждого реферала
- Vercel: реферальная программа

**Пример:** 10 000 подписчиков Telegram + 2-3 нативных поста в месяц = 30–100k руб/мес.

**Плюсы:** пассивный доход, лиды на другие продукты.  
**Минусы:** медленно строится, нужен объём, зависимость от алгоритмов.

## Модель 4: Инфопродукты

Пакетируешь знания в курс, книгу, шаблоны, гайды.

**Что продаёшь:** знания в структурированном виде.

**Форматы:**
- PDF/гайд: 500–5000 руб разово
- Мини-курс (текстовый, Telegram): 2000–15000 руб
- Видеокурс: 10 000–100 000+ руб
- Шаблоны и промпты: 500–3000 руб (но объём)

**Плюсы:** продаёшь один раз, получаешь много раз; высокая маржа.  
**Минусы:** нужна аудитория доверяющая тебе, быстро устаревает в AI.

**Для вайб-кодера:** шаблоны CLAUDE.md, промпты, готовые решения — востребованы и быстро создаются.

## Комбинирование моделей

Лучшая стратегия — комбинировать:

```
Контент → аудитория
Аудитория → лиды на консалтинг (высокий чек)
           → подписчики SaaS
           → покупатели инфопродуктов
```

Пример роста:
1. **Месяц 1–3**: делаешь контент, берёшь первый заказ на консалтинг
2. **Месяц 3–6**: монетизируешь кейсы в инфопродукты
3. **Месяц 6–12**: запускаешь SaaS-инструмент для своей аудитории

## Ценообразование: главная ошибка

Начинающие занижают цены потому что «недостаточно хороши».

**Правило:** цена определяется ценностью для клиента, не твоими усилиями.

Если клиент экономит 20 часов в месяц по 5000 руб/ч = 100 000 руб → инструмент за 5000 руб/мес — выгодная сделка для него, даже если ты сделал его за 2 дня.

**Метод ценообразования:**
1. Посчитай ценность для клиента (экономия времени × их ставка)
2. Возьми 10–20% от этой суммы
3. Это твоя цена

## Первые деньги быстро

Хочешь денег за 2 недели — консалтинг. Алгоритм:
1. Определи нишу (e-commerce, стройка, медицина — не «всё подряд»)
2. Сформулируй 1 конкретный результат (не «AI-консультации», а «настрою AI-ассистента для обработки заявок за 3 дня»)
3. Напиши 5–10 людям из этой ниши напрямую с конкретным предложением
4. Проведи первый проект, сделай кейс
5. Повтори

---

::: info Главное
Любая модель работает, если есть реальная ценность для клиента и ты умеешь её донести. Начни с самого быстрого пути (консалтинг), накопи кейсы, потом масштабируй через продукт.
:::
