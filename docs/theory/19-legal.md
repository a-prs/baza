# Юридика для AI-продукта

Скучно, но важно. Без минимального юридического оформления — риск блокировки аккаунта, жалоб, проблем с маркетплейсами. Разберём что нужно, и как закрыть это быстро.

## Что нужно обязательно

| Документ | Зачем | Критичность |
|---|---|---|
| Политика конфиденциальности | Требует App Store, Google Play, Telegram, большинство рекламных платформ | Обязательно |
| Пользовательское соглашение | Описывает правила использования, ограничивает ответственность | Важно |
| Оферта / описание услуги | Если берёшь деньги | Обязательно при оплате |
| Cookie-уведомление | EU/GDPR (если есть пользователи из ЕС) | Нужно для EU |

## Политика конфиденциальности

Самый важный документ. Должна отвечать на вопросы:
- Какие данные собираешь?
- Как используешь?
- Кому передаёшь?
- Как долго храниш?
- Как пользователь может удалить свои данные?

### Шаблон-минимум для Telegram-бота

```markdown
# Политика конфиденциальности бота [Название]

Последнее обновление: [дата]

## Что мы собираем

При использовании бота мы получаем:
- Telegram ID, имя пользователя и имя профиля
- Текст сообщений, которые вы отправляете боту
- Дату и время взаимодействий

## Как используем

Данные используются исключительно для:
- Предоставления функций бота
- Технической поддержки
- Улучшения сервиса

Мы не продаём и не передаём данные третьим лицам,
за исключением сервисов, необходимых для работы бота
(OpenAI/Anthropic для AI-функций).

## Хранение

Данные хранятся на сервере в [страна]. Время хранения: [N дней/навсегда до удаления].

## Ваши права

Вы можете запросить удаление своих данных, написав боту команду /delete
или на email: [твой email].

## Контакты

[email или Telegram-контакт]
```

### Где разместить

Для Telegram-бота — минимум один из вариантов:
1. Команда `/privacy` → бот присылает текст
2. URL в описании бота (BotFather → Edit Bot → Edit About → ссылка на страницу)
3. HTML-страница на GitHub Pages или Vercel

```python
@dp.message(Command("privacy"))
async def cmd_privacy(message: Message):
    await message.answer(
        "🔒 Политика конфиденциальности\n\n"
        "Бот получает ваш Telegram ID и текст сообщений для предоставления AI-ответов. "
        "Данные не передаются третьим лицам (кроме Anthropic API для AI-функций). "
        "Для удаления данных: /delete\n\n"
        "Полный текст: https://your-site.com/privacy"
    )

@dp.message(Command("delete"))
async def cmd_delete(message: Message):
    await db.delete_user(message.from_user.id)
    await message.answer("Ваши данные удалены из нашей системы.")
```

## Пользовательское соглашение (ToS)

Ключевые пункты:

```markdown
# Пользовательское соглашение

## Использование сервиса

Используя бота, вы соглашаетесь:
- Не использовать его для незаконных целей
- Не пытаться взломать или перегрузить сервис
- Принимать, что AI-ответы могут содержать ошибки

## Ограничение ответственности

Сервис предоставляется «как есть». Мы не несём ответственности за:
- Точность AI-ответов
- Потери от использования информации от бота
- Временную недоступность сервиса

## Изменения

Мы можем изменить условия. Продолжение использования = согласие с новыми условиями.
```

## Если берёшь деньги

### Самозанятый (РФ)

Самый простой вариант для физлиц: платишь 4% с доходов от физлиц, 6% от юрлиц. Оформляется через приложение «Мой налог».

Что нужно:
- Зарегистрироваться как самозанятый в «Мой налог»
- Для Telegram Stars: вывод через Fragment → проверь налоговый статус криптодохода в твоей юрисдикции
- Для ЮКасса/Stripe: можно как физлицо (ЮКасса требует ИП или ООО для некоторых типов) или как самозанятый

### Оферта для SaaS-подписки

```markdown
# Публичная оферта

Настоящий документ является публичной офертой [ИП/Самозанятый Имя].

## Предмет договора

Предоставление доступа к AI-ассистенту [название] в соответствии с выбранным тарифом.

## Тарифы

- Бесплатный: 5 запросов в день
- Базовый: 200 Stars/месяц — 200 запросов в день

## Порядок оплаты

Оплата через Telegram Stars. Возврат средств: Stars не возвращаются согласно политике Telegram.

## Срок действия

Подписка активна в течение оплаченного периода.
```

## Данные пользователей (GDPR/ФЗ-152)

Если есть пользователи из ЕС → GDPR. Если хранишь персональные данные граждан РФ → ФЗ-152.

### Минимальный compliance

```python
# 1. Собирать только необходимое (Data Minimization)
# Плохо: сохранять всю историю чатов навсегда
# Хорошо: хранить только то что нужно для функций, с TTL

# 2. Хранить в РФ (ФЗ-152) если есть пользователи РФ
# VPS в РФ или EU — смотря на аудиторию

# 3. Команда /delete — обязательна
@dp.message(Command("delete"))
async def delete_my_data(message: Message):
    await db.execute("DELETE FROM users WHERE telegram_id = ?", (message.from_user.id,))
    await db.execute("DELETE FROM messages WHERE user_id = ?", (message.from_user.id,))
    await message.answer("Все ваши данные удалены. Спасибо за использование сервиса.")

# 4. Не передавать данные лишним третьим лицам
# При использовании Claude/OpenAI: пользователи должны знать
# что их сообщения обрабатываются AI-сервисом
```

## AI-специфичные риски

### Ответственность за AI-контент

Добавь в ToS:

```
AI-ответы генерируются автоматически и могут содержать неточности, 
устаревшую информацию или ошибки. Не используйте их как профессиональный 
совет (медицинский, юридический, финансовый) без проверки у специалиста.
```

### Авторские права на AI-контент

Сейчас спорная зона. Безопасная позиция: не заявляй что AI-контент полностью твой оригинальный, особенно для публикации.

### Запрещённые юзкейсы

Anthopic и OpenAI имеют свои Usage Policies — нарушение = блокировка API-ключа. Запрещено:
- Генерация вредоносного контента
- Обман пользователей что они общаются с человеком
- Сбор данных для манипуляций

Если продаёшь доступ к AI — твоя ответственность контролировать использование.

## Быстрый старт: минимальный набор

```
Неделя 1 (обязательно):
□ Написать политику конфиденциальности (используй шаблон выше)
□ Разместить её по URL
□ Добавить команду /privacy в бота
□ Добавить команду /delete

Неделя 2 (если берёшь деньги):
□ Зарегистрироваться как самозанятый
□ Написать оферту
□ Добавить ссылку на оферту перед первым платежом

Позже (если растёт аудитория):
□ Юридическая консультация (2-3 часа, ~10-15к₽)
□ Полноценный ToS
□ GDPR compliance если есть EU-пользователи
```
