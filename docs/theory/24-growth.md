# Каналы роста для AI-продукта

После Product-Market Fit встаёт вопрос: откуда брать пользователей системно? Разбираем каналы которые реально работают для indie-продуктов и небольших команд.

## Принцип одного канала

Ошибка: пытаться делать SEO + контент + таргет + партнёрства одновременно.

Правило: сфокусируйся на одном канале до тех пор пока он не даёт стабильный поток пользователей — потом добавляй следующий. Один канал, освоенный до результата, лучше пяти «попробованных».

## Канал 1: Контент (SEO + SMM)

**Подходит:** B2C с длинным циклом принятия решения, обучение, инструменты.

**Telegram-канал** — самый доступный старт для русскоязычной аудитории:

```
Структура контента:
- 40% образование (как делать X)
- 30% инсайты (что узнал, что попробовал)
- 20% кейсы (что получилось у клиентов)
- 10% продажи (явная реклама продукта)

Частота: 3-5 постов в неделю
Формат: короткий (до 1000 символов) + длинный (разборы) чередуй
```

**SEO статьи** — работает для инструментов с поисковым спросом:

```
Проверь спрос: Wordstat, Google Keyword Planner
Примеры хороших запросов для AI-инструментов:
- «как автоматизировать [задача]»
- «чат-бот для [ниша]»
- «AI [конкретная задача]»

Статья которая работает:
1. Заголовок = запрос
2. Конкретный практический контент (не реклама)
3. Бесплатный инструмент / шаблон на скачивание
4. CTA в конец: «Попробуй в нашем боте»
```

## Канал 2: Сообщества (Community-led)

Участие в чужих сообществах — быстрее и дешевле чем строить своё.

```
Тактика:
1. Найди 5-10 Telegram-чатов / Discord-серверов где сидит твоя ЦА
2. Неделю отвечай на вопросы, помогай — не продавай
3. Укажи в профиле или подписи чем занимаешься
4. Когда кто-то спрашивает о проблеме которую ты решаешь — дай полезный ответ + упомяни бота

Что НЕ делать:
- Не спамить ссылку сразу после входа в чат
- Не постить «запустил бота, посмотрите»
- Не делать рекламные посты в чатах без договорённости с админом
```

## Канал 3: Партнёрства (B2B2C)

Самый быстрый способ добраться до большой аудитории — через уже существующий канал.

```
Модели партнёрства:
1. Дополнение к чужому продукту
   Пример: «AI-плагин для [популярный инструмент]»
   
2. Co-marketing: совместный контент
   Пример: вебинар/статья вместе с другим автором/брендом
   
3. Интеграция через API
   Пример: твой AI в чужой CRM/сервисе

4. Affiliate: они рекомендуют — ты платишь %
```

Как найти партнёров: кто уже работает с твоей ЦА, но не конкурирует?

## Канал 4: Холодный аутрич (B2B)

Работает для дорогих B2B-продуктов (>5000₽/мес).

```python
# Структура холодного письма / сообщения
"""
Тема: [конкретная проблема которую решаешь]

Привет [имя],

Заметил что вы [контекст — что конкретно ты знаешь о них].

Мы помогаем [ЦА] [результат] — например, [конкретный кейс или число].

Это актуально для вас? Готов показать на 15-минутном звонке.
"""

# Метрики холодного аутрича
# Open rate: 40-60% (хороший)
# Reply rate: 5-15% (хороший)  
# Conversion to call: 2-5% (нормальный)
```

Ключевое: персонализация. Не «привет, у меня есть бот», а «привет, вижу вы нанимаете операторов поддержки».

## Канал 5: Платный трафик

Запускать ПОСЛЕ валидации органических каналов. Таргет усиливает то что уже работает, не заменяет его.

```
VK Ads (для РФ B2C):
- ЦА: интересы + look-alike с первых клиентов
- Бюджет: 5000₽ тест → если CAC < LTV/3, масштабировать

Telegram Ads:
- Рекламирует боты в чужих каналах
- Мин. бюджет: $500 (высокий порог)
- Хорошо для широкой аудитории

Контекст (Яндекс):
- Для продуктов с поисковым спросом
- Запросы = проблема, не решение («как автоматизировать отчёты», не «AI-бот для отчётов»)
```

## Метрики для выбора канала

| Канал | Время до результата | Стоимость | Масштаб |
|---|---|---|---|
| Контент/SEO | 3-6 месяцев | Низкая | Высокий |
| Сообщества | 2-4 недели | Минимальная | Средний |
| Партнёрства | 1-3 месяца (договориться) | Низкая | Высокий |
| Холодный аутрич | 2-4 недели | Время | Ограничен |
| Таргет | 1-2 недели (тест) | Высокая | Высокий |
| Рефераллы | 1-2 месяца (накопить пользователей) | Низкая | Умеренный |

## Как выбрать первый канал

```
Задай три вопроса:
1. Где моя ЦА проводит время онлайн?
2. Что требует меньше ресурсов на старте?
3. В чём у меня есть преимущество (пишу хорошо / много знакомых из ниши / есть связи)?
```

Пример:
- Пишу хорошо + аудитория в Telegram → Telegram-канал + сообщества
- B2B продукт → холодный аутрич + партнёрства
- Есть бюджет → таргет (но только после органической валидации)

## Retention важнее привлечения

Самая дорогая ошибка — привлекать пользователей в дырявое ведро.

```
Нормальный D7 retention для AI-инструментов: 30-40%
Плохой: < 20% (нужно сначала улучшить продукт)
Хороший: > 50%

Перед масштабированием привлечения:
□ D7 retention > 30%
□ 3-5 платящих пользователей органически
□ NPS или ручная обратная связь: понятно почему пользователи остаются
```
